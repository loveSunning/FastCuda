# Prompt Files

Prompt files are task entrypoints, not global policy.

Recommended mapping:

- `implement-gemm-kernel.md`: add or revise GEMM kernels
- `implement-flashattention-kernel.md`: add or revise FlashAttention kernels
- `optimize-kernel.md`: optimize a known kernel bottleneck
- `benchmark-kernel.md`: run or design benchmark work
- `investigate-environment.md`: diagnose toolchain or GPU visibility issues
